# Asteroids Game

A simple asteroids game made with Python and Pygame.

## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/your-repository.git
    ```
2.  Install the dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3.  Run the game:
    ```bash
    python main.py
    ```

## Controls

*   **W**: Move forward
*   **S**: Move backward
*   **A**: Rotate left
*   **D**: Rotate right
